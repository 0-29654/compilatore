C++ COMPITI ALUNNO

1. Sul PC docente avvia C++ Visual Base, scheda Compiti alunni, e premi Avvia server.
2. Comunica agli alunni IP:porta e codice sessione.
3. Sul PC alunno inserisci i dati, compila e invia.
4. Windows Firewall può chiedere di consentire la rete privata: autorizzare solo la rete del laboratorio.
5. Per compilare serve g++ (MSYS2/MinGW) nel PATH.

Per creare l'installer esegui CREA_INSTALLER.bat su Windows con .NET 8 SDK e Inno Setup 6.
